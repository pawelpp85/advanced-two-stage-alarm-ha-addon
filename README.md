# PawelPP85 Home Assistant Add-ons

Custom Home Assistant add-on repository.

## Add-ons

### Advanced Two-Stage Alarm

Path: `advanced_two_stage_alarm/`

Features:
- two-stage alarm flow (warning -> main alarm),
- immediate-trigger entities,
- profile-based trigger sets,
- rich trigger-reason text for UI and TTS,
- WS communication with Home Assistant.

## Add Repository in Home Assistant

Use this URL in Supervisor -> Add-on Store -> Repositories:

`https://github.com/pawelpp85/advanced-two-stage-alarm-ha-addon`

## Version

Current beta: `0.0.1b2`
